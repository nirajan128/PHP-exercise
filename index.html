A PHP script that converts a temperature (input1) from Fahrenheit to Celsius if input2 is true, and from Celsius to Farenheit if input2 is false.

a PHP script that takes a student's numerical grade as input1 and prints the corresponding letter grade based on the following scale:

  • 90 and above: A
  • 80 to 89: B
  • 70 to 79: C
  • 60 to 69: D
  • Below 60: E

APHP script that takes an integer as input1 and determines if it is even or odd. Print the result accordingly.

A PHP script that calculates the ticket price for a movie based on the age (input1) of the viewer

  • Children (ages 0 to 5): Free
  • Youth (ages 6 to 17): $8
  • Adults (ages 18 to 64): $12
  • Seniors (ages 65 and above): $10

A PHP script that checks if a given year (input1) is a leap year or not. A leap year is divisible by 4 but not by 100, except when it is divisible by 400.

a PHP script that takes an integer (1 to 7) as input1 and prints the corresponding day of the week, where 1 represents Monday, 2 represents Tuesday, and so on.

a PHP script that calculates and prints the factorial of a given positive integer (input1).

When the button is clicked, an AJAX call will happen that asks your PHP program, for 5 "dice" values. These dice are random values, simulating a “roll” of 5 dice with 6 sides each. Your program will output HTML code that the AJAX call will display. Each die roll should be in a separate HTML element . Include BootStrap classes in your output to help make it more interesting and appealing. Use rand(1,6) to generate the random integers

In addition to the dice rolling feature of the dice rolling app, change the output so that any of the dice that roll a 1 are shown in red with white text. Have all other dice follow a different colour scheme of your choice. Hint: You may assume that BootStrap will be loaded client side.

Create a PHP script that creates a string containing 5 comma separated temporary passwords, and provides the string back to the AJAX client-side call. The temporary passwords should each be a random mix of upper- and lower-case letters and digits of length 16. (HINT: Use rand for random numbers and the chr function to convert an ASCII code to a string. See http://www.asciitable.com/ for ASCII codes.)
